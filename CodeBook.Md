*** Code Book

** Labeling
The data set has been labeled with appropriately with the following transformations:

(1) "Acc" -> "Acceleration"

(2) "Mag" -> "Magnitude"

(3) "Gyro" -> "Gyroscope"

(4) "^t" -> "Time"

(5) "^f" -> "Frequency"

(6) "-mean()" -> "Mean"

(7) "-std()" -> "SD"


** Data transformation
The dity dataset includes average values of each variable for each activity and each subject.
